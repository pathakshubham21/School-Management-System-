What's New in Training and Certification





New Digital Course: Alexa for Developers


This free, two-hour online digital course introduces you to the Alexa Skills Kit (ASK) and covers core skills, ASK Software Development Kit (SDK) concepts, and ASK integration with other AWS services.


New Digital Course: Data Analytics Fundamentals


Data Analytics Fundamentals, which replaces the digital course Big Data Technology Fundamentals, will teach you the latest on how to seamlessly plan a data analysis solution using the suite of AWS services.


New Version of AWS Security Fundamentals Digital Course Now Available

AWS Security Fundamentals (Second Edition) is now available. This introductory digital course features updated, streamlined content to help you better understand security in the cloud for your organization.


New and Updated Exam Readiness Courses


Built by the experts at AWS, new Exam Readiness courses for Professional-level and Specialty certifications are now available. Take free digital training or sign up to attend a class.
